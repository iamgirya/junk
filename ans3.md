# 1) Читаемость и структура

* **Виды/разрезы**: разрез **E–E** обозначен, но нужно убедиться, что на линии сечения стоят стрелки направления и подпись по ГОСТ 2.305 (знак линии сечения, стрелки, буквы «E» у стрелок). Сейчас в PDF читаются лишь буквы «E», без уверенной фиксации стрелок — добавьте/уточните графику, чтобы исключить двусмысленность. 
* **Таблицы**: на поле есть ссылки «**См. табл.**». Проверьте, что таблица действительно присутствует на листе и содержит все параметры, на которые идут отсылки (диаметры/глубины/PCD/кол-во/тип обработки). Если таблица вынесена на отдельный лист, дайте **сквозную ссылку** с номером листа/таблицы. Сейчас по тексту ссылка есть, содержимое таблицы не читается — заполнение нужно подтвердить и привязать. 
* **Десятичный разделитель**: используйте **запятую**, а не точку (ГОСТ 2.304). В тексте видны значения вида «67.862» — в оформленном КД должно быть «67,862». 

# 2) Полнота размеров для изготовления

* **Центральное отверстие под М6**: в разрезе/видах встречается размер **⌀6,8**. Для унификации с узлом под винты **М6** примите **⌀6,5** (клиренс М6) с допуском по задаче (например, Н12 либо «по ГОСТ 30893.1, класс m», если размер свободный). Проставьте **фаски входа 1×45°** с обеих сторон. Это приведёт деталь к одному уровню с плитами, где под М6 используется ⌀6,5. 
* **Трёхлучевой узор отверстий**: запись «**3,5 120° 3 отв.**» требует конкретизации:

  1. если это **сквозные отверстия ⌀3,5**, задайте **допуск на диаметр** (например, ⌀3,5 H12),
  2. задайте **расположение** — **диаметр делительной окружности (PCD)** или радиальный размер от оси **A**,
  3. если отверстия **зенкованные/потайные**, оформите «**⌀D×90°**» (D — наружный диаметр зенковки) и **глубину/обеспечение заподлицо**; одной записи «90°» недостаточно, требуются **D и/или глубина**. Сейчас видна только «3,5… 120°… 3 отв.» и отдельно «90°» — этого мало для обработки. 
* **Ступени и переходы**: для всех цилиндрических поясков укажите **номиналы Ø и предельные отклонения**. Если пояски функциональные (посадка во втулку/в гнездо), назначьте **поле допуска** (например, **h9/h10** с обоснованием). Для свободных — оставить под общие допуски (30893.1, класс m). 
* **Фаски**: замените одиночные углы «45°» на формат **a×45°** (например, 0,5×45°) у всех кромок, где требуется снятие фаски. Укажите на виде/разрезе у конкретной кромки, не только в общем примечании. 
* **Галтели**: задано «**Неуказанные переходные скругления R0,4**» — хорошо, но для **напряжённых переходов ступеней** рекомендуем проставить **конкретные R** (например, R0,8…R1) непосредственно на разрезе, чтобы исключить острые переходы. 
* **Глубины отверстий**: если есть **глухие** отверстия, обязательно задать **глубину** и **минимальный остаток**; для сквозных — это указание не требуется. По текущему листу глубины для 3×⌀3,5 не читаются — добавьте, если они глухие. 

# 3) Соответствие ГОСТ по допускам формы/расположения

* **Базы**: на чертеже присутствуют **A** и **B** — сохранить: **A** — ось детали, **B** — опорный торец. Привязки к базам используйте во всех **рамках геометрических допусков**. Сейчас в техтребованиях присутствует «**Общий позиционный допуск отверстий 0,3 мм**», но **не указан(ы) баз(ы)**; необходимо дописать **«отн. A|B»** (или привязать к конкретной базовой схеме в рамке). Иначе общий допуск неработоспособен для контроля. 
* **Локальные геодопуски**: на видах встречается обозначение «**0,1 A**» без читаемого символа (OCR скрывает знак). Уточните **тип** допусков:

  * для опорного торца **B** — **перпендикулярность 0,1 к A** (символ ⟂ в рамке),
  * для наружных цилиндрических поясков — **радиальное биение ⌀0,1 к A** (символ биения и знак «⌀» в рамке).
    Запишите по ГОСТ 2.308 (ISO 1101), с указанием **базы A** и нанесением рамок у соответствующих размеров. Сейчас без символа тип допуска неочевиден. 
* **Общие допуски**: в ТТ уже есть:
  «**ГОСТ 30893.1, класс m**»,
  «**ГОСТ 30893.2-K**»,
  «**Острые кромки притупить 0,3×45°**»,
  «**R0,4 неуказанные**»,
  «**Общий позиционный допуск отверстий 0,3 мм**».
  Оставьте эту структуру, но пункт с позиционным допуском перепишите как:
  **«Неуказанные отверстия: позиционный допуск ⌀0,3 относительно A|B»**. Это приведёт запись к контролепригодной.

## Итоговые правки (краткий чек-лист для внесения на лист)

1. Заменить **⌀6,8 → ⌀6,5** под М6; добавить фаски **1×45°** с обеих сторон. 
2. По группе **3×⌀3,5, 120°**: добавить **допуск на диаметр**, **PCD/радиус** расположения, **начальный угол/привязка к базе**, для потая — **⌀D×90°** и/или глубину.

4. В техтребованиях пункт про **позиционный допуск ⌀0,3** дописать «**отн. A|B**». 
5. Поставить **локальные Ra** (1,6/0,8) на опорном торце и посадочных Ø. Общая Ra 3,2 — оставить. 
6. Указать **материал** и **покрытие** (анодирование для Al или антикор для стали). 
7. Проверить наличие/заполнение **таблицы**, на которую ссылаются «См. табл.», и корректные **стрелки линии сечения E–E**.
