Ниже — перечень того, что на чертеже «Стакан» необходимо **добавить/уточнить/исправить** для однозначного изготовления и контроля по ЕСКД/ГОСТ. (Анализ основан на содержимом присланного PDF «АМТ430.2301.15.12.008.02 Стакан». ) 

# 1) Размеры и поля допусков (ГОСТ 2.307; ГОСТ 25346/25347)

* Для **всех функциональных диаметров** (наружные Ø, внутренние Ø, посадочные по сопряжению) — указать **поля допусков** (например, H7/h6, js7 и т.п.) вместо «голых» номиналов. Сейчас часть диаметров дана **без поля допуска**, что не позволяет обеспечить посадку. (Применять систему допусков и посадок **ГОСТ 25346/ГОСТ 25347**.) 
* Для **общей длины** и **ступенчатых участков** (L, длины буртов/утонений) — назначить **предельные отклонения** (симметричные или односторонние) по **ГОСТ 30893.1** (ISO 2768) вместо отсутствующих либо подразумеваемых. Рекомендуется класс **m** (или другой, исходя из точности узла), явно записанный в техтребованиях. 
* Для **канавок/проточек** (если присутствуют в разрезе) — задать **ширину b, глубину t, радиусы сопряжений**, **поля допусков** на b и t. Если канавка предназначена под уплотнение/стопорное кольцо — указать **тип/нормаль** (стандарт на профиль). Сейчас геометрия канавки не полностью определена. 
* Для **фасок** — указывать **размер × угол** (например, 0,5×45°). Одна лишь запись «45°» **без линейной величины** фаски недопустима. 
* Для **сквозных/глухих отверстий** (если есть) — указать **глубину** и, при глухих отверстиях, **минимальный остаток** (d∙min) либо высоту неполной резьбы; при резьбе — см. п.4. 

# 2) Геометрические допуски (ГОСТ 2.308; ISO 1101)

* Ввести **датумную схему**:
  **A** — ось базового наружного цилиндра (или внутреннего, если он функционально базовый),
  **B** — опорный торец (основной установочный), при необходимости **C** — второй торец/посадочная поверхность. 
* Назначить:

  * **Соосность** (или **радиальное биение**) внутренней поверхности относительно базы **A** — для обеспечения концентричности стенок/посадок.
  * **Перпендикулярность** торца **B** к оси **A** — под установку/упор.
  * При наличии ступеней — **биение** буртов/посадочных шеек относительно базы **A**.
    Запись и рамки допусков — по **ГОСТ 2.308**. Сейчас геометрические требования отсутствуют. 

# 3) Шероховатость поверхностей (ГОСТ 2.309; ГОСТ 2789/ГОСТ ISO 1302)

* Указать **общую шероховатость** для неответственных поверхностей (например, **Ra 3,2**), а для **посадочных/уплотнительных** — **более строгие Ra** (например, **Ra 1,6 / 0,8** по необходимости).
* Там, где **обработка не требуется**, поставить знак **«без снятия материала»** (треугольник без полки) или «не обрабатывать» — по ГОСТ 2.309. Сейчас часть поверхностей **без обозначения шероховатости**. 

# 4) Резьбы и входы (если применимо) (ГОСТ 2.307; ГОСТ ISO 965)

* Для **внутренних резьб** — полное обозначение **M…–7H**, **количество отверстий**, **глубина резьбы**, **диаметр сверления** (если дан сверлёж) и **фаска входа** (например, 1×45°).
* Для **наружных резьб** — поле допуска (например, **6g**), длина нарезки и фаска/снятие захода.
  Сейчас в зоне предполагаемой сверловки/резьбы не хватает **глубин и входных фасок**. 

# 5) Радиусы/галтели и притупления (ГОСТ 2.307)

* На переходах диаметров задать **радиусы галтелей** (например, **R0,5…R2** — по расчёту прочности/технологии).
* Для **всех острых кромок** — типовое требование «**Острые кромки притупить 0,2–0,5×45°; заусенцы удалить**». Сейчас притупления кромок явно **не заданы**. 

# 6) Материал, состояние, покрытия (ГОСТ 2.105; ГОСТ 9.xxx)

* Указать **материал** с нормативом (например, **Сталь 20 ГОСТ 1050** / **АМг6 ГОСТ 4784** и т.п.) и, при необходимости, **состояние/термообработку** (НВ/HB, Т…/закалка, отпуск). Сейчас **материал не прочитан** из чертежа. 
* Если изделие **алюминиевое** и требуется защита — задать **анодирование**: **вид/класс/толщина**, допуски на изменение размеров, **маскировку** резьб/посадок (ГОСТ 9.301/9.302/9.303). Если **стальное** — указать **фосфатирование/цинк/окраску** или консервацию. На текущем листе покрытие **не указано**. 

# 7) Общие допуски: корректная запись (ГОСТ 30893.1; ГОСТ 30893.2)

* В техтребованиях задать:
  «**Неуказанные предельные отклонения линейных и угловых размеров — по ГОСТ 30893.1, класс *m* (или другой по необходимости).**»
  «**Неуказанные допуски формы и расположения — по ГОСТ 30893.2, класс *K*.**»
* **Не* смешивать** эту запись с полями **H12/h12/IT…** (они применяются **к конкретным размерам** по ГОСТ 25346/25347). Сейчас общая запись на листе отсутствует/нечитабельна — следует **ввести корректно**. 

# 8) Базы, разрезы, сечения, выносные элементы (ГОСТ 2.305; 2.307)

* **Обозначения сечений/разрезов** (например, **E–E**) должны иметь: стрелки направления, шрифтовые обозначения, привязку по ГОСТ 2.305, а на виде — полную **привязку размеров к видимым контурам** (без «висящих» размеров). Проверить и при необходимости **уточнить маркировку сечения** и расположение размеров на разрезе. 
* Ввести **базовые обозначения** на видах/разрезах (A, B, C) и использовать их в рамках геометрических допусков (см. п.2). Сейчас **баз нет**. 

# 9) Дополнительно по оформлению основной надписи (ГОСТ 2.104; 2.109)

* Проверить **обозначение документа**, **наименование детали («Стакан»)**, **массу** (допустимо «для справок»), **материал**, **масштаб**, **стадию**, **лист/листов** — заполнение по ГОСТ 2.104/2.109; при отсутствии данных — **дополнить**. На текущем PDF часть полей читается фрагментарно — требуется приведение к норме.

## Рекомендуемый блок «Технические требования» (пример)

1. Неуказанные предельные отклонения размеров — по **ГОСТ 30893.1**, класс **m**. Неуказанные допуски формы и расположения — по **ГОСТ 30893.2**, класс **K**.
2. База **A** — ось наружного цилиндра; база **B** — опорный торец.
3. Соосность внутреннего Ø относительно базы **A** — ⌀0,0X; перпендикулярность торца **B** к оси **A** — 0,0X (значения по расчёту).
4. Шероховатость: **Ra 3,2** — если не указано иначе; для посадочных поверхностей — **Ra 1,6**/**0,8**.
5. Острые кромки притупить **0,2–0,5×45°**; заусенцы удалить.
6. Материал: **<указать марку и норматив>**; покрытие: **<тип, норматив, толщина, маскировка>** (при необходимости).
7. Резьбы (если есть): **M…–7H(6g)**, глубина резьбы … мм; фаска входа **1×45°**.

> После добавления перечисленных размеров, полей допусков и указаний, чертёж «Стакан» будет соответствовать требованиям ЕСКД и обеспечит однозначность изготовления и контроля в производстве.
